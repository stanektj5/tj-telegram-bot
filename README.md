# TJ Telegram Bot

Personal AI assistant via Telegram.

## Commands

- /workout - today's workout
- /groceries - current list + reminder
- /add [item] - add to grocery list
- /bought - mark purchased, reset timer
- /markets - top market events today
- /ai - latest AI and Claude news
- /brief - full morning briefing

## Setup

1. Fill in .env with real values
2. pip install -r requirements.txt
3. python bot.py

## Deploy

Push to GitHub, connect to Railway.app, add env vars in Railway dashboard.
